# Spring Boot Project Management System
A production-grade backend system built using Spring Boot, designed to simulate enterprise-level project and employee management workflows.

## 🚀 Overview

This project demonstrates:
- Clean architecture (Controller → Service → Repository → Integration)
- Salesforce REST integration
- JWT Authentication (planned)
- DTO-based API design
- Scalable and modular code structure

---

## 🏗️ Architecture

Controller → Service → Mapper → Repository → Database  

External Integrations (Salesforce, Web-based apps, etc.)

---

## ⚙️ Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- MySQL
- REST APIs
- Salesforce Apex Integration
- Maven
